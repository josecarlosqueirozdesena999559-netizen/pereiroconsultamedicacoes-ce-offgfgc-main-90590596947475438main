# ConsultMed Pereiro

Aplicacao web para consulta de medicamentos e informacoes das UBS de Pereiro-CE.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Capacitor

## Desenvolvimento local

Requisitos:

- Node.js 18+
- npm

Passos:

```sh
git clone <URL_DO_REPOSITORIO>
cd <PASTA_DO_PROJETO>
npm install
npm run dev
```

## Build de producao

```sh
npm run build
```

Os arquivos gerados ficam em `dist/`.

## Deploy

O deploy deve seguir o fluxo configurado para este repositorio e para a hospedagem do projeto.

## Edicao de codigo

Voce pode editar o projeto localmente na sua IDE, pelo GitHub web ou por Codespaces, conforme preferir.
